# Quote Saver

Quote Saver is a web application that generate and save quotes. It is built using Next.js, a React framework that enables server-side rendering and static site generation. The application also uses Tailwind CSS for styling and Vercel for deployment.

## Getting Started

First, run the development server:

```bash
npm run dev

npm install

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Open [https://github.com/konynunez/quote-saver.git](https://github.com/konynunez/quote-saver.git) git repository
